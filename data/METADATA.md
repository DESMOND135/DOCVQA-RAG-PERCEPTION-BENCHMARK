
# Dataset Source & Citation

The evaluation results in this project are based on the **DocVQA (Document Visual Question Answering)** dataset.

## Official Citation
> Tito, M., Karatzas, D., Valveny, E. (2020). DocVQA: A Dataset for Document Visual Question Answering. WACV 2021.

## Source Link
[www.docvqa.org](https://www.docvqa.org/)

## Repository (HuggingFace)
[lmms-lab/DocVQA](https://huggingface.co/datasets/lmms-lab/DocVQA)

---
*Note: This data folder contains a subset of 50 validation samples converted to TXT and 5 representative source images for qualitative audit.*
